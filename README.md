OŠK Kamenná Poruba — Webový koncept

Prehľad

Tento projekt predstavuje návrh (koncept) webovej stránky futbalového klubu OŠK Kamenná Poruba. Slúži na ukážku štruktúry, rozloženia obsahu a navigácie moderného klubového webu. Ide o statický frontend bez napojenia na databázu alebo backendové služby.

Účel projektu

* demonštrovať návrh používateľského rozhrania a navigácie
* prezentovať spôsob zobrazovania informácií o klube, hráčoch a zápasoch
* simulovať reálne sekcie športového webu

Projekt nie je produkčné riešenie — obsah má ilustračný charakter.

Funkcionalita (koncept)

* navigácia medzi hlavnými sekciami stránky
* základné informácie o klube
* prehľad hráčov a kategórií
* zoznam plánovaných zápasov
* galéria (ukážková štruktúra)
* kontaktná sekcia
* simulované prihlasovacie rozhranie

Stav projektu

Koncept / prototyp určený na prezentáciu návrhu a štruktúry webu.
